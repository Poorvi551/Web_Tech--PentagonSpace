Started with basic structure of HTML.

<!DOCTYPE html> ---->  DOCUMENT type version of HTML-5
<html> ---> Root Element
  <head> ---> meta information
     <meta>  ---> browser information
     <title>DOCUMENT</title> ---> tab data
  </head>
  <body>  ---> All the elements are present
    HELLO WORLD HTML!
  </body>
</html>

Tags:-

They are of two types -
1. Paired - They have both opening and closing tags.
   - Bold tag  --> <b>
   - Strong tag  --> <strong>
   - Italic tag  --> <i>
   - Emphasis tag --> <em>
   - mark tag    --> <mark>
   - Subscript tag --> <sub>
   - Superscript tag --> <sup>
   
2. Unpaired - They are self closing tags.
   - Paragraph tag --> <p>
   - Pre formatted tag --> <pre>
   - Image tag --> <img>
